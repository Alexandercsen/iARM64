<h1>-@INFO-</h1>

Hello, welcome to the iARM64 project! This project started as a way to look productive while i studied in my University's Computer Science Centre, as there
was nobody to tutor. It was originally just your standard notes taking app and not OS specific or anything.

As the note editor progressed, it quickly gained the ability to compile C code, because the world doesnt need just another notes taking Application!
So here i was with a C code editor that i wrote in C, with a C IDE (CLion) that i would use more often than a stupid simple code editor, no matter how lightweight it was.
(As a Systems dev, i get it. As a general programmer, i do not. Efficiency ends for me once it becomes a pain to use.)

So, then came the switch to ARM64 Assembly for MacOS, as you dont see as many of them, and i didnt see one i liked personally.
In this case, i'd rather this than using VScode with a plugin or something. It makes a lot more sense.
Also, feel free to steal the code, make a new branch. Its completely open source. I just wanted to write some basic assembly.

This will be updated pretty consistently i hope with more features, such as file management!

Features:

-Rendered in SDL2.
<br>
-Text Uses TTF.
<br>
-Compiles using Clang (Apple native compiler for Assembly).
<br>
-
everything else is pretty self explanitory.


<h1>-@HOWTO-</h1>

This is very simple. I intend to eventually package this and add a download button. Alas thats a few major updates away so, in the meantime!
All you have to do is download this repo and run the program, save the Assembly code, it will create a file (in version 2, you will be allowed rename files and open them up at whim, so you arent just writing over the same file again and again.)

There you go, there it is
